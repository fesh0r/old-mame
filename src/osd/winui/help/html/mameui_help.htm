Help Menu

       Help Topics :  Selecting this option from the help menu will launch the MameUI help file.

       Troubleshooting :  Selecting this option from the help menu will display the troubleshooting topic in the help file.

       New Game Changes :  Selecting this option from the help menu will display added, removed, changed games.

       Mame Whatsnew.txt :  Selecting this option from the help menu will display the baseline Mame whatsnew.txt file which contains information about source changes, games added, driver improvements, etc.

       About : Selecting this option from the help menu will display information about the version of MameUI and its contributors.
