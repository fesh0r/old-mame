View Menu

       Toolbar : Selecting this option from the view menu will toggle the toolbar on or off.

       Status Bar : Selecting this option from the view menu will toggle the status bar on or off.

       Folder List : Selecting this option from the view menu will toggle the folder list view on or off.

       Screenshot : Selecting this option from the view menu will toggle the screenshot area on or off.

       Screenshot Tab : Selecting this option from the view menu will toggle the screenshot area tabs, allowing to choose between the multiple different art types.  Note that history.dat information is only displayed in the snapshot view.

       Fullscreen : Selecting this option from the view menu will switch the GUI into full screen mode.  This will expand to fill the current desktop resolution, it will hide the menu bar, and it will hide the windows taskbar.  Toggle in and out of it with F11, other alt shortcut keys are still active.

       Show Pictures : Selecting this option from the view menu will allow toggling the display of the individual art tabs in the screenshot area.

       Large Icons : Selecting this option from the view menu will display large icons in the gamelist area.

       Small Icons : Selecting this option from the view menu will display small icons in a different grid pattern in the gamelist area.

       List : Selecting this option from the view menu will display small icons in the gamelist area.

       Details : Selecting this option from the view menu will display small icons along with full game name and any other fields chosen with the View • Customize Fields option.

       Grouped : Selecting this option from the view menu will display small icons along with full game name and any other fields chosen with the View • Customize Fields option, however the clone games will be displayed indented and alphabetized under their parents..

       Arrange Icons : Selecting this option from the view menu will order the gamelist area by the choice made; game, directory, manufacturer, times played, type, or year.

       Line Up Icons : Selecting this option from the view menu will re-align the icons in views where it does not automatically happen.

       Refresh : Selecting this option from the view menu will initiate an update of the gamelist while MameUI searches for the existence of zip files that match game directory names.  This is not as exhaustive as the audit function on the File menu.

       Custom Filters : Selecting this option from the view menu will allow for the selective hiding of certain types of games.  Manually typing in a phrase in the filter text box will only display games that have that text in their names [note valid input for the description filter are, game name, driver name e.g. 'cps2.c', and manufacturer].

       Customize Fields : Selecting this option from the view menu will allow choosing which fields are displayed in columnar mode in the game list details view, selectable from the View • Details.  The columns can also be ordered by moving them up and down the right hand side of the well.  The game name field is always the first item and cannot be changed.
