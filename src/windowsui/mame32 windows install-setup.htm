Installation and Setup in Windows 95/98/Me/NT/2000

Follow these steps to set up MAME32 in Windows 95/98/Me/NT/2000:

1. Download the MAME32.ZIP zipfile. Click here to find locations for downloading the MAME32 emulator.

2. Create a directory/folder on your hard drive (i.e. c:\mame32)

3. Open this directory and copy the MAME32.ZIP zipfile into it

4. Using an archive program like WinZip or Pkunzip, extract the contents of the MAME32.ZIP (ensure you use the -d option in WinZip or Pkunzip to get recursed subdirectories). In WinZip select Extract from the Actions menu and be sure the 'Use Folder Names' checkbox is checked.

5. Place your game ROMs (readily available in convenient zip format) into the sub-directory called \mame32\roms which you may need to create (NOTE: game ROMs are *not* distributed with MAME32)

6. Place your sound sample zip files into the sub-directory called \mame32\samples which you may need to create (NOTE: sample zips will have the same name as the game ROMs)

7. Place your icons .zip file into the \mame32\icons directory.

8. Place your large images screenshot file combined from the multiple ‘snap.zip’ files into the \mame32\snap directory.

9. Place contents of the background pack in \mame32\snap. Rename the one you want to use as the background to bkground.png.

10. You have the option to add flyers.zip and cabinets.zip into the \mame32\flyers and \mame32\cabinets directories respectively. Flyers.zip is a file composed of .bmp, or .png flyer art. Cabinets.zip is a file composed of .bmp, or .png images of the actual arcade machines. These files are maintained by third party sites (see the Directories topic for links).

11. You can also grab the history.dat file Here. This will give you game information and history on particular games. Place the history.dat file in the \mame32 directory along w/ the mame32.exe. The history will only show for games that have a history entry [Donkey Kong for instance] and it will only show while in screenshot mode.

12. Double click MAME32.EXE, select your game and press play!

Storing ROM and Sample files:

You should store the ROM and Sample files in the zipped file within the \mame32 directory structure. If you wish to unzip the ROM or Sample files, make sure you unzip the files to the correct directory underneath the \mame32 directory structure. Following is the search order that MAME32 will look for ROM and Sample files:

1. \mame32\’gamename’ directory

2. \mame32\’gamename.zip’ zip file

3. \mame32\roms\’gamename’ directory and \mame32\samples\’gamename’ directory

4. \mame32\roms\’gamename.zip’ zip file and \mame32\samples\’gamename.zip’ zip file
